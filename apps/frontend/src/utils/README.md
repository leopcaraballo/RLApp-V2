# Utilities

Funciones utilitarias agnósticas del dominio.

Ejemplos:
- api-client.ts: Cliente HTTP configurado con interceptores
- formatters.ts: Formateo de fechas, moneda, teléfono
- validators.ts: Funciones de validación reutilizables
- constants.ts: Constantes globales
- helpers.ts: Funciones helper generales
- date-utils.ts: Manejo de fechas
- string-utils.ts: Manipulación de strings

Cada función debe ser pura y testeable.
