# Threat Model

- broken access control
- exfiltracion de token o sesion desde el browser de staff
- suscripcion realtime no autorizada a vistas operativas internas
- exposicion de PII en display
- exposicion de PII en payload realtime o snapshots de monitor fuera del boundary aprobado
- perdida de eventos por falla de mensajeria
- abuso de rebuild o endpoints sensibles
