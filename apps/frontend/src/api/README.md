# API Routes

Route handlers de Next.js para mocks o proxy a backend (opcional).

Estructura:
- /api/auth/ - Endpoints de autenticación
- /api/queue/ - Endpoints de colas
- /api/transactions/ - Endpoints de transacciones

NOTA: En producción, estos deberían apuntar a un backend .NET
Aquí solo se incluyen como referencias o mocks de desarrollo.
