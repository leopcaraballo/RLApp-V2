# Reception Module Components

Componentes del módulo de recepción.

Responsabilidades:
- Registrar nuevos clientes
- Crear tickets de cola
- Asignar servicios
- Mostrar estado de cola

Componentes esperados:
- NewCustomerForm
- TicketCreationForm
- QueueStatus
- ServiceSelector

Todos mapean a UC (Use Cases) y S (Especificaciones) en `/docs/project`.
