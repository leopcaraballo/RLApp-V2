# Custom Hooks

Hooks React reutilizables que encapsulan lógica de la aplicación.

Ejemplos esperados:
- useAuth.ts: Contexto y estado de autenticación
- useApi.ts: Cliente HTTP con interceptores
- useRealtime.ts: Conexión WebSocket/SignalR
- useNotification.ts: Sistema de notificaciones
- useDebounce.ts: Debounce de valores
- useFetch.ts: Obtención de datos con loading y error

Cada hook debe ser independiente y reutilizable.
